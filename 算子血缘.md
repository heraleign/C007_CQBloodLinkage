### 3.2 字段连线规则

| 场景 | 连线样式 | 说明 |
|------|---------|------|
| 直接映射（1对1） | 实线 `────` | 源字段直接赋值给目标字段 |
| 多字段合并（N对1） | 多条实线汇聚 `╮────` | 多个源字段经计算得到目标字段 |
| 条件分支（CASE WHEN） | 虚线 `- - -` | 条件逻辑，非全量流入 |
| 跨表关联字段 | 带颜色标注 | 不同源表用不同颜色区分 |

### 3.3 表卡片字段展示规则

```
┌─────────────────────────────────┐
│  表名                  [系统标签] │
│  🟡 层级标签   类型标签           │
├─────────────────────────────────┤
│  [类型图标] 字段名   [约束标签]   │  ← 有血缘关系的字段优先展示
│  [类型图标] 字段名   [约束标签]   │
│  [类型图标] 字段名                │
│  ─────────────────────────────  │
│  全部 N 列  ∨                    │  ← 点击展开全部字段
└─────────────────────────────────┘
```

**字段类型图标：**

| 图标 | 含义 |
|------|------|
| `#` | 数值型（INT/DECIMAL） |
| `Aa` | 字符串型（VARCHAR/STRING） |
| `📅` | 日期时间型 |
| `⚑` | 布尔型 |

**字段约束标签：**

| 标签 | 含义 | 颜色 |
|------|------|------|
| `PK` | 主键 | 蓝色 |
| `FK` | 外键 | 橙色 |
| `NN` | 非空 | 灰色 |

## 四、算子级血缘展示设计

### 4.1 触发方式

用户**点击目标表中任意字段**，在画布下方或右侧弹出「字段口径解析」面板。

### 4.2 算子面板设计

```
┌──────────────────────────────────────────────┐
│  字段口径解析                [复制] [展开] [×]  │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                              │
│  目标字段：V_RES . hdl_org                                                   │
│                ┌─ 参与字段 ──────────────────────────────────────────────────────────┐    │
│  │                                                │    │
│  │  来源表              字段名          类型      角色                  │    │
│  │  ─────────────────────────────────────────────────────────────────  │    │
│  │  🟡 VT_BUS          hdl_org         Aa        条件判断字段           │    │
│  │  🟡 VT_BUS          item_id         #         过滤条件字段           │    │
│  │  🟡 VT_BUS          bal             #         过滤条件字段           │    │
│  │                                                                     │    │
│  └─────────────────────────────────────┘    │
│                                                                              │
│  ┌─ 加工逻辑（SQL表达式）──────────────────────────────────────────────┐    │
│  │                                                                     │    │
│  │  CASE                │    │
│  │    WHEN  VT_BUS.hdl_org = '06'  THEN  '187'                        │    │
│  │    WHEN  VT_BUS.hdl_org = '07'  THEN  '601'                        │    │
│  │    WHEN  VT_BUS.hdl_org = '08'  THEN  '981'                        │    │
│  │    ELSE  '168'                                                      │    │
│  │  END  AS  hdl_org                                                   │    │
│  │  WHERE                                                              │    │
│  │    VT_BUS.item_id  NOT LIKE  '2%'                                   │    │
│  │    AND  VT_BUS.bal  <>  0                                           │    │
│  │                                                                     │    │
│  └─────────────────────────────────────┘    │
│                                                                              │
│  ┌─ 算子类型标注 ──────────────────────────────────────────────────────┐    │
│  │  🔀 条件映射（CASE WHEN）    🔍 过滤（WHERE）                        │    │
│  └─────────────────────────────────────┘    │
│                                                                              │
│  解析来源：AST解析  置信度：96%   最后更新：2026-05-20                        │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 4.3 算子类型体系

| 算子类型 | 图标 | 触发场景 | 示例 |
|---------|------|---------|------|
| 直接赋值 | `→` | `a.col AS col` | `user_id → user_id` |
| 条件映射 | `🔀` | `CASE WHEN` / `IF` | 枚举值转换 |
| 聚合计算 | `∑` | `SUM/COUNT/AVG/MAX/MIN` | `SUM(amount)` |
| 字符串处理 | `Aa` | `CONCAT/SUBSTR/TRIM` | 字段拼接 |
| 数值计算 | `±` | 四则运算 | `amount * tax_rate` |
| 日期处理 | `📅` | `DATE_FORMAT/DATEDIFF` | 日期格式转换 |
| 过滤条件 | `🔍` | `WHERE` 中引用的字段 | 参与过滤但不直接输出 |
| 关联键 | `🔗` | `JOIN ON` 中的字段 | 关联条件字段 |
| 自定义函数 | `⚙️` | `UDF` 调用 | 业务自定义函数 |

### 4.4 画布联动高亮

点击目标表字段后，画布同步高亮：

```
高亮规则：
  ① 目标字段所在行：蓝色背景高亮
  ② 所有参与该字段计算的源字段：橙色高亮
  ③ 连接这些字段的连线：加粗 + 对应颜色
  ④ 不相关的字段和连线：降低透明度（opacity: 0.2）

示例效果：
  VT_BUS                    V_RES
  ┌──────────────┐          ┌──────────────────┐
  │ item_id      │ ·····→   │ item_id          │  ← 灰色（不相关）
  │ dat_org      │          │ dat_src          │  ← 灰色
  │ bal          │ ──────→  │ frs_lol_bbk      │  ← 灰色
  │ hdl_org 🟠   │ ══════→  │ hdl_org  🔵      │  ← 高亮（选中字段）
  └──────────────┘          └──────────────────┘↑
  BRMD_C_T84_VBS
  ┌──────────────┐
  │ brn_org_id   │  ← 灰色（不参与hdl_org计算）
  └──────────────┘
```

## 五、交互流程设计

### 5.1 完整交互流程

```
用户进入血缘页面
    │
    ▼
默认展示「表级血缘」视图
    │
    ▼
点击顶部切换「字段级」
    │
    ▼
画布展开各表字段列表
连线从表级连线细化为字段级连线
    │
    ├─→ 用户点击目标表某字段（如 hdl_org）
    │       │
    │       ▼
    │   ① 该字段行高亮（蓝色）
    │   ② 参与计算的源字段高亮（橙色）
    │   ③ 相关连线加粗高亮
    │   ④ 底部/右侧弹出「字段口径解析」面板
    │   ⑤ 面板展示：参与字段列表 + SQL表达式 + 算子类型
    │├─→ 用户点击「展开全部列」
    │       │
    │       ▼
    │   表卡片展开显示所有字段
    │   画布自动重新布局
    │
    └─→ 用户点击空白处
            │
            ▼
        取消高亮，恢复默认状态
```

### 5.2 字段搜索定位

```
┌─────────────────────────────────────────────────────┐
│  🔍 搜索字段名...                                    │
└─────────────────────────────────────────────────────┘│
    ▼
输入字段名 → 高亮所有包含该字段名的节点
           → 自动滚动画布到第一个匹配位置
           → 显示匹配数量（如：找到 3 处）
```

## 八、设计要点总结

| 设计决策 | 理由 |
|---------|------|
| 字段级连线从表卡片字段行直接引出 | 直观体现字段对应关系，避免歧义 |
| 算子面板点击触发而非常驻 | 减少画布信息密度，按需展示 |
| 参与字段区分「计算字段」和「过滤字段」 | 帮助用户理解字段的不同参与角色 |
| SQL表达式语法高亮展示 | 降低阅读成本，快速定位关键逻辑 |
| 画布联动高亮 + 非相关内容降透明度 | 聚焦用户注意力，减少干扰 |
| 变更感知标注在字段行内 | 第一时间感知上游变化，辅助影响分析 |
